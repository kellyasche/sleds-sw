Summary of SAT data

Lets check to see what percentage of the master dataset has taken the SAT.

There is a tiny percentage of the dataset that has taken the SAT, .5%. With this information I don’t see a need to check whether there is a significant difference in students taking the SAT based on RUCA category or County. Even if there is a significant difference, there just isn’t enough observations to make a difference.


sat.total <- master.8 %>%
  group_by(sat.taken) %>%
  summarize(n = n()) %>%
  ungroup() %>%
  mutate(pct = n / sum(n))

datatable(sat.total, class = "cell-border stripe", filter = "top", rownames = FALSE,
          options = list(columnDefs = list(list(className = "dt-center", targets = 0:1))))  %>%
  formatPercentage(3, digits = 1) %>%
  formatCurrency(2, digits = 0, "")
write_csv(master.8, "Data/SLEDS/Masters/Master-8.csv")
